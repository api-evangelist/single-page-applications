XSPAN is an internet-based cross-species anatomy network designed to support access to tissue-based genetic information.It provides interoperability between databases of key model embryos which, in turn, facilitates access to gene expression and other tissue-based data across species.XSPANs database of cross-species anatomy information can be accessed programmatically via SOAP calls.
